# QEWS Vite Prototype

Run it with:

```bash
npm install
npm run dev
```

## Includes
- React + Vite
- Tailwind CSS v4
- Framer Motion
- Lucide icons
- Interactive world map with clickable sites
- Slider-based simulation lab
- Team recruiting section and interest form

## Notes
The map is a local asset generated from a public-domain style world outline and is already included in `src/assets`.
